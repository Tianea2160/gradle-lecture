## Introduce
Gradle에 관해 세미나를 하기 위해서 만들어진 실습 코드 저장소 입니다.
해당 저장소는 멀티 모듈 프로젝트를 연습하기 위한 최소한의 코드가 들어가 있습니다.

## Enviroment
- language : java 17
- gradle : 8.0.0
